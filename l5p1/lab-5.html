Lab 5

Problema 1: Exemplificati utilizarea polimorfismului creing urmatoarele clase:

class Forma {

   public:

      virtual double ComputeArea() = 0;

      virtual const char * GetName() = 0;

};

class Dreptunghi : public Forma {

      int Latime,Lungime;  

   public:

      Dreptunghi(int lungime,int latime);

      double ComputeArea();

      const char * GetName();

};

class Cerc : public Forma {

      int Raza;  

   public:

      Cert(int raza);

      double ComputeArea();

      const char * GetName();

};

class Triunghi : public Forma {

      int X1,Y1,X2,Y2,X3,Y3;  

   public:

      Triunghi(int x1,int y1,int x2,int y2,int x3,int y3);

      double ComputeArea();

      const char * GetName();

};

In functia main creati un vector de pointeri catre instate de tipul Forma si asignatile cu diverse obiecte de tipul Dreptunghi, Cerc sau Triunghi. Dupa acest lucru apelati pe rand functia ComputeArea pentru fiecare element din vector si afisati numele formei (functia GetName), respectiv aria ei. Functia GetName va returna numele clasei (pentru clasa Dreptunghi va returna "Dreptunghi", pentru clasa Cerc va returna "Cerc", etc.


Problema 2: Realizati un sistem de plug-in-uri cu urmatoarea arhitectura:
  • Un proiect de tipul executabil (.exe) care sa faca urmatoarele actiuni:
    • citeste de la tastatura numele unui fisiere librari (*.dll)
    • Incarca acel fisier dinamic (LoadLibraryA din Windows API)
    • Verifica daca fisierul incarcat exporta 2 functii: GetName si Compute
    • Daca da - obtine 2 pointeri catre aceste functii
    • Cere de la tastatura doua numere de tipul int
    • Afiseaza pe ecran GetName, urmat de rezultatul de la Compute(primul numar citit,al doilea numar citit)
  • Un al doilea proiect de tipul librari (.dll)
    • Acest proiect trebuie sa contina doua functii cu urmatorul prototip:
      • const char* GetName() {...}
      • int Compute(int,int)
    • Sa creeze un fisier *.def in care sa exporte cele doua functie
  • Proiectul de tipul librarie (dll) trebuie facut de 4 ori (petru functiile adunare, scadere, inmultire, impartire). Mai exact, functia Compute(int,int) va returna altceva in functie de proiectul in care a fost implementata (Adunare, Scadere, ...)
Pentru utilizarea functiilor API - utilizati  #include "windows.h"
Acest proiect este optional si se recomanda utilizarea Visual Studio pentru realizarea lui (2010 / 2013).
